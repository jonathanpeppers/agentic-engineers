Agentic Engineers

We are a group of software engineers at Microsoft exploring how AI agents can transform the way we build, review, and ship software.

🏢 Microsoft · Developer Division · .NET

What We Do

Agentic Engineers is a team within Microsoft's Developer Division, working across the .NET ecosystem. We build frameworks and tools used by millions of developers — including .NET MAUI, .NET for Android, .NET for iOS, and related libraries.

We are actively integrating AI-powered agents into our engineering workflows: from Copilot-authored pull requests to automated code review powered by multi-model adversarial systems. This site tracks our progress and shares what we learn along the way.
