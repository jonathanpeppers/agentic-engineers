# Agentic Engineers

Documents and links from our group, "Agentic Engineers", at Microsoft.

> **Microsoft Employees:** You can also view the internal wiki page on how to join: [Agentic Engineers Wiki](https://dev.azure.com/devdiv/DevDiv/_wiki/wikis/DevDiv.wiki/50016/Agentic-Engineers)

## Agent Frameworks

* [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) — Microsoft's framework for building AI agents
* [Model Context Protocol — C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) — official C# SDK for building MCP servers and clients, maintained with Microsoft
* [.NET Extensions](https://github.com/dotnet/extensions) — includes AI/Copilot integration points

## Skills & Plugins

* [Anthropic's Skills](https://github.com/anthropics/skills) — public repository of agent skills for Claude Code
  * [`skill-creator` skill](https://github.com/anthropics/skills/tree/main/skills/skill-creator) — a skill that helps you create, test, and optimize new skills
* [Anthropic Claude Plugins (official)](https://github.com/anthropics/claude-plugins-official) — Anthropic-managed directory of high-quality Claude Code plugins
* [.NET Skills](https://github.com/dotnet/skills) — skills to assist AI coding agents with .NET and C#
* [.NET Arcade Skills](https://github.com/dotnet/arcade-skills) — reusable AI skills, plugins, and agents for dotnet product repos
* [Larry's Agent Plugins](https://github.com/lewing/agent-plugins) — curated agent skills marketplace
* [Stéphane's MAUI Copilot Plugins](https://github.com/StephaneDelcroix/maui-copilot-plugins) — includes a .NET MAUI Appium skill
